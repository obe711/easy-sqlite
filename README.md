# Easy SQLite

A simple JSON SQLite library

## Installation

First install [Node.js](http://nodejs.org/). Then:

```sh
$ npm install easy-sqlite-json
```

## Importing

```javascript
// Using Node.js `require()`
const EasySqlite = require("easy-sqlite-json");

// Using ES6 imports
import EasySqlite from "easy-sqlite-json";
```
